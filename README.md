# Chat App

Chat app with Basic functionalities using Flutter and FireBase.

## Installation

```bash
flutter pub get
```

## Usage

```bash
 flutter run
```

## Features

- Firebase cloud based chat system.
- Sign-Up & Login Features (With Profile Photo)
- Auto Logout when Session ends
